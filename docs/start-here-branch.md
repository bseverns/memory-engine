# Start Here Branch

This branch is the practical entry trunk of the manual.

Use it when you need to get oriented quickly, bring up a machine, rehearse the current lifecycle, or prove that a node is ready for real use.

## Branch Logic

This branch is deliberately mixed.

- some pages are for first contact with the machine
- some are for running it
- some are for proving the current stack is ready

That mix is intentional because these are the pages most likely to be opened on install day or immediately before a public session.
