# Steward & Participant Materials Branch

This branch is the handoff trunk of the manual.

Use it when the question is less about architecture and more about what a steward or participant actually needs in front of them.

## Read This Branch By Situation

| Situation | Read this first | Then go to |
|---|---|---|
| a participant needs plain-language guidance before or after recording | [participant-prompt-card.md](./participant-prompt-card.md) | [start-here.md](./start-here.md) |
| a steward needs the shortest recovery ritual | [OPERATOR_DRILL_CARD.md](./OPERATOR_DRILL_CARD.md) | [maintenance.md](./maintenance.md) |
| a steward needs the fuller operational context | [maintenance.md](./maintenance.md) | [AT_A_GLANCE.md](./AT_A_GLANCE.md) |

## Branch Logic

This branch stays intentionally small.

- it is not a broad training curriculum
- it is not a moderation handbook
- it is the human-facing edge of the machine

If this branch starts growing too fast, that is usually a sign that the operator surface or the public ritual needs simplification upstream.
