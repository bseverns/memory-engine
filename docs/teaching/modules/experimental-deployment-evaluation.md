# Module: Experimental Deployment And Evaluation

## Purpose

Teach how to evaluate Memory Engine deployments using bounded, reproducible evidence.

## Anchor Reading

- [experimental-proofs.md](../../experimental-proofs.md)
- [roadmap.md](../../roadmap.md)
- [V1_3_STATUS.md](../../V1_3_STATUS.md)

## Key Ideas

- Evaluation should test operational truth, participant comprehension, and stewardship continuity.
- Pilot evidence should be structured and comparable across rehearsals.
- Claims of readiness require restore and handoff rehearsal evidence.

## In-Class Flow (35-55 min)

1. Choose one proof item from the proof board.
2. Define success/failure signals before running any activity.
3. Record outcomes with a template and identify next validation pass.

## Reflection Prompts

- Which questions can be answered in a classroom simulation?
- Which questions require live pilot context?
