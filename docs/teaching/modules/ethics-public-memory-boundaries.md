# Module: Ethics, Public Memory, And Research Boundaries

## Purpose

Make project boundaries explicit so evaluation does not drift into extraction.

## Anchor Reading

- [MISSION_EXPANSION.md](../../MISSION_EXPANSION.md)
- [ARCHIVE_STEWARDSHIP.md](../../ARCHIVE_STEWARDSHIP.md)
- [participant-prompt-card.md](../../participant-prompt-card.md)

## Key Ideas

- Bounded scope is an ethical choice, not missing ambition.
- Participant agency requires clear consent language and real revocation pathways.
- Research value can come from steward evidence and public comprehension without surveillance expansion.

## In-Class Flow (30-45 min)

1. Review one tempting “feature expansion” idea.
2. Evaluate whether it deepens Memory Engine or widens it.
3. Rewrite the idea into a bounded experiment or defer it.

## Reflection Prompts

- What does this project refuse, and why?
- Where does stewardship replace data extraction?
