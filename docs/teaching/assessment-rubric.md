# Lightweight Assessment Rubric

Use this rubric for course checkpoints, steward practicums, or pilot rehearsals.

## Scale

- **1 - Emerging**: incomplete understanding, frequent guessing
- **2 - Developing**: partial understanding, needs prompts
- **3 - Proficient**: reliable independent execution
- **4 - Steward-ready**: clear, repeatable, and teachable to others

## Rubric Dimensions

| Dimension | 1 - Emerging | 2 - Developing | 3 - Proficient | 4 - Steward-ready |
|---|---|---|---|---|
| Lifecycle literacy | Cannot trace artifact states end-to-end | Traces most states with gaps | Accurately traces ingest, playback, revocation, expiry | Explains lifecycle plus failure consequences and edge cases |
| Surface role clarity | Confuses kiosk/room/ops boundaries | Understands roles but mixes APIs/actions | Correctly maps each surface purpose | Can teach role boundaries and rationale clearly |
| Consent/revocation explanation | Uses vague or incorrect language | Mostly correct but misses caveats | Gives participant-safe explanation of modes/revocation | Gives clear explanation under pressure and checks comprehension |
| Ops recovery practice | Cannot complete drill without intervention | Completes drill with step-by-step prompting | Completes drill and verifies state after actions | Completes drill, documents evidence, and briefs next steward |
| Evaluation rigor | Notes are anecdotal and unstructured | Notes include observations but weak criteria | Uses templates and bounded metrics consistently | Produces clean pilot evidence with explicit limits and next actions |
| Boundary discipline | Proposes widening/surveillance features casually | Notices boundaries but inconsistently applies them | Stays within project mission and constraints | Defends boundaries and reframes ideas without platform drift |

## Evidence Artifacts

A learner/team is steward-ready when they can produce:
- one completed lifecycle trace lab
- one consent/revocation audit note
- one successful ops recovery drill log
- one participant comprehension check summary
- one handoff-ready steward note packet
