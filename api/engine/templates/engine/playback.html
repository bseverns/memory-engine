Listening notes

How this surface behaves

Shared mix

Many kiosks, one room Any recording station pointed at this node feeds the same archive, and this surface listens back through that combined pool.

Windows

Moving source slices Long recordings can be revisited as shorter windows, so the room hears different edges of the same memory over time.

Weathering

Patina, not collapse Repeatedly heard material picks up softness and grain without being crushed into caricature.

Pacing

Composed drift The loop moves through arrival, gathering, weathering, and release rather than staying in a flat shuffle.

Use

Keep it dedicated This machine does not record. Leave capture to /kiosk/ and let this surface stay with the room.

Startup

One tap if needed If the browser blocks autoplay after boot, tap Start listening once and the room can remain hands-off after that.
