Bring-up and recovery first

These are the first actions to reach for during install day, daily open, or a rough reboot. Keep the living controls and routing checks above the archive summaries.

Start-of-day ritual 1. Open `/ops/` and confirm the state badge below makes sense. 2. Run the output tone. 3. Run live monitor with closed headphones. 4. Open `/kiosk/` and `/room/` before the public arrives.
Say this during live monitor "Room Memory check. One quiet line. One normal line. One clap or finger snap." This gives you breath noise, normal voice, and a transient in one pass.
If reboot breaks the kiosk button Check Chromium focus before touching firmware. A restore bubble, permission chip, or browser chrome after boot is more likely than Leonardo drift.
